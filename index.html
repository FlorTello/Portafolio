Proyectos Personales


Aplicación móvil YAPE


Información del Proyecto

Este proyecto consistió en realizar el una simulación de proceso de regitro de la apliación Movil del Banco de Credito del Perú. Donde el usuario ingresa solo necesita aceder a la aplicación e ingresar su número para empezar el registro.

Requerimientos Proyecto: El trabajo con componentes fue muy importante y el uso de la API que nos bridarón para relizar el almacen de datos.

Objetivos Proyecto: La reto fue que el proyecto se culminaría en 2 y medío cumpliendo todos requerimientos.

Aplicación responsive Labcar


Información del Proyecto

Este proyecto consistió en replicar la página web de la empresa LabCar su funcionalidad principal es que se trace una ruta entre el ubicación actual y un punto de destino ingresado por el usuario y que estime la tarifa en base a la distancia entre los puntos.

Requerimientos Proyecto: El uso del framework de css Bootstrap fue una las herramientas y también contar con el apoyo de la API de Google Maps para el trazado de rutas.

Objetivos Proyecto: Lograr el diseño para movil y desktop con la API de Google Maps.

Labcar es una empresa de taxis que te permite pedir un taxi cuando quieras y en donde quieras. Se ha comprobado que Labcar tiene las tarifas más bajas del mercado. Sabemos que tu tiempo es dinero, por lo que si nuestro servicio se demora en llegar más de lo establecido te daremos un 10% de descuento en tu viaje.
